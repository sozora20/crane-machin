# Crane Machine — Кранный автомат

## Описание
Веб-игра «Кранный автомат» — точная копия аркадного claw machine. Управляй краном, захватывай шары и выигрывай призы!

## Возможности
- Реалистичная физика шаров (гравитация, столкновения, отскоки)
- 22 шара в 10 цветах с неоновыми эффектами
- Анимированный кран с захватом
- Система призов с настраиваемой вероятностью
- История всех игр
- Тёмная неоновая тема
- Адаптивный дизайн
- Админ-панель с паролем

## Запуск локально
1. Клонировать репозиторий
2. `npm install`
3. Скопировать `.env.example` → `.env.local` и заполнить
4. `npx prisma migrate dev`
5. `npm run dev`

## Деплой на Vercel
1. Подключить репозиторий в Vercel
2. Создать Neon Postgres через Vercel Integrations
3. Добавить env variables: `DATABASE_URL`, `ADMIN_PASSWORD`, `ADMIN_TOKEN_SECRET`
4. Deploy

## Переменные окружения
- `DATABASE_URL` — строка подключения Neon Postgres (с pgbouncer)
- `ADMIN_PASSWORD` — пароль для входа в /admin
- `ADMIN_TOKEN_SECRET` — секрет JWT (минимум 32 символа)
